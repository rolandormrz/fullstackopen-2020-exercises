Full stack open 2020

This course is an introduction to modern web development with JavaScript, with an emphasis on building single page applications with ReactJS that use REST APIs built with Node.js. There is also a section on GraphQL. 

The course is split up into nine parts and the completed exercises for each part are contained within each folder in this repository.

Part 1: Introduction to React
Part 2: Communicating with server
Part 3: Programming a server with NodeJS and Express
Part 4: Testing Express servers, user administration
Part 5: Testing React Apps
Part 6: State management with Redux
Part 7: React router, custom hooks, styling app with CSS and web pack
Part 8: GraphQL
Part 9: Typescript
